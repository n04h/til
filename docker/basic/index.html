    nginxのコンテナを起動させる

    1
    docker run -d -p 80:80 --name {コンテナ名} nginx

    nginxを80番ポートで起動してくれる。 docker imageがローカルで見つからない場合は、Docker Hubからpullしてくれる。

    コンテナ一覧を見るとき

    1
    docker container ls

    -aオプションをつけると起動していない状態のコンテナ一覧を見れる

    コンテナを止めるとき

    1
    docker container stop {コンテナ名}

    コンテナを起動するとき

    1
    docker container start {コンテナ名}`

    コンテナを削除するとき

    1
    docker container rm {コンテナ名}

    イメージを削除するとき

    1
    docker image rm {IDもしくは image name}
